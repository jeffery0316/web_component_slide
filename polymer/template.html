HTML Template

使用javascript直接操作template 
    $('#useme').on('click', function () {
        var content = document.querySelector('template').content;

        // 更新 template DOM 中的内容，並加至span中
        var span = content.querySelector('span');
        span.textContent = parseInt(span.textContent) + 1;
        $('#container').html('Template used: <span>' + span.textContent + '</span>');
    });

Custom Element




  • 定義自己的element, 每個element能擁有自己的properties, methods, events
  • 其他framework能夠一起並用
  • 直接與底層的DOM操作, addEventListener...等function仍然可以重複使用
  • 鍵盤, focus, ARIA (accessibility)
  • chrome/firefox devtools可以直接偵測element tag
  • 許多現成的kit ＠ http://customelements.io

Custom Element - Example


        <element name="x-foo" constructor="XFoo">
          <section> Foo bar test</section>
          <script>
            var section = this.querySelector('section');
            this.register({
              prototype: {
                readyCallback: function() {
                  this.textContent = section.textContent;
                },
                foo: function() {alert('foo() is called');}
              }
            });
        </element>

Use Custom Element

      <x-foo></x-foo><!-- names must contains dash -->
      /* or use in DOM */
      var elem = document.createElement('x-foo');
      elem.addEventListener('click', function(e) {
        e.target.foo(); // alert 'foo() called'.
      });
      /* new a object */
      var elem = new XFoo();
      document.body.appendChild(elem);
      /* define element in JS */
      var XFoo = document.registerElement('x-foo');
      document.body.appendChild(new XFoo());

Shadow DOM

  • DOM/CSS scoping，element並不能被直接存取
  • 可以做data-binding
  • 封裝DOM Tree，確保其他library不會更動到其中的element

Shadow DOM

  • Shadow Root

      • 與shadow root相連的element都稱為shadow host

      在進行頁面渲染時，僅顯示shadow root 的內容。shadow host的值會被shadow root覆蓋。

      但是使用javascript取textContent，則只會取得shadow host的值 (shadow root被封裝)

  • Shadow DOM Tree
  • Shadow DOM with document fragment
    •           <button>Hello, world!</button>
          <script>
          var host = document.querySelector('button');
          var root = host.createShadowRoot();
          root.textContent = 'こんにちは、影の世界!';
          </script>

HTML Imports

  • 元件化方法：

    • 用iframe區隔不同的document，但是iframe與當前頁面的互相操作不方便，且iframe的width/height也很複雜

    用Ajax插入HTML DOM，並產生JS+HTML的骯髒寫法

      <!DOCTYPE html>
      <html>
        <head>
          <link rel="import" href="x-foo.html">
        </head>
        <body>
          <x-foo></x-foo> <!-- Element definition is in x-foo.html -->
        </body>
      </html>

Polymer - Data binding

Polymer supports two-way data binding. Data binding extends HTML and the DOM APIs to support a sensible separation between the UI (DOM) of an application and its underlying data (model). Updates to the model are reflected in the DOM and user input into the DOM is immediately assigned to the model. 

      <polymer-element name="tag-name" constructor="TagName">
        <template>
        <!-- shadow DOM here -->
        </template>
        <script>Polymer('tag-name');</script>
      </polymer-element>
